# 🎮 8-Bit Vertical Runner Game (Pygame)

A simple 8-bit style vertical runner game built using Python and Pygame.

---

## 📌 About the Project

This game demonstrates:

- 2D Game Development using Pygame
- Collision Detection
- Random Enemy Spawning
- Score System
- Game Loop
- Menu System
- Increasing Difficulty

---

## 🕹 Game Rules

- Player can move only **UP** and **DOWN**
- 🔴 Red block = Enemy (Game Over if hit)
- 🔵 Blue block = Score block (Increases score)
- Speed increases as score increases
- Player must stay inside the screen
